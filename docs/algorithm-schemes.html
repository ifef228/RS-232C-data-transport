Сначала подключают приёмник, затем передатчик. На каждый кадр с надёжной доставкой приёмник отвечает кадром ACK с тем же номером seq.

2. Алгоритм узла-передатчика (передача одного файла)

Блок «Надёжно» — вызов процедуры отправки кадра с ожиданием ACK (см. рис. 3). Цикл по фрагментам закодированных данных повторяется, пока не переданы все байты.
